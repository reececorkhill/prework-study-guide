HTML

  • !DOCTYPE is not case sensitive because it is a declaration.
  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • Inline CSS is when a developer wants to style an element directly in the HTML file.
  • Internal CSS is when you add your CSS within the style element inside the head element.
  • External CSS is when you add your CSS to a separate file and link this to the HTML file.

Git

  • git status: This checks what branch we are currently on
  • git checkout -b branch-name: This creates a new branch and switches to it
  • git commit -m "...": This commits your staged snapshot as a new commit snapshot.
