HTML

  • HTML is used to create the structure of a webpage. It is made up of elements which work together to provide content and the basic framework for all pages on the web.

    • Elements

    • Text Elements
    • Image Elements
    • Header Elements
    • List Elements

    Elements that are nested inside other elements are called child elements. If there are three elements nested inside of a body element, those three nested elements
    are known as child elements. The element that holds all the children—in this case, the body element is known as the parent element. This “parent/child” language
    is often used by developers to explain the relationship among elements in HTML.

Git

  • git status (Shows modified files in working directory, staged for your next commit).
  • git commit -m “[description]” (Commit your staged content as a new commit snapshot).
  • git checkout (This command is to move the working branch to a new branch).
    • -b (This is a flag that creates a new branch (b = branch)
    • example (git checkout -b feature/starter-code)
  • git add (This command adds modifications in the current working branch to the staging area).
    • -A (This flag indicates that we want to add all changes. The staging area informs Git that the changes will be added in the next commit).
  • git pull (This command will receive a branches modifications into the local environment)
    • origin (This keyword indicates the source of the pull in the GitHub Repo).
    • main (This keyword indicates the branch. In our case, we are receiving a branch called main from the GitHub repository).
    • Example (git pull origin main) - This pulls the branch modifications to the local environment and tells git that the source is in the GitHub repo and we’re receiving a branch called main.
